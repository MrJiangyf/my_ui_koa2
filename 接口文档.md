1. 注册：用户名是否存在

   ```javascript
   let api = "/api/user/isExist"; 
   let type = "post"; 
   let query = {userName: ""};
   let result = {
      code: 200,
      msg: "用户名不存在",
      data: {
        userName: "",
        nickName: "",
        picture: "",
        city: "",
      }
   }
   ```

2. 注册用户

   ```javascript

   ```
